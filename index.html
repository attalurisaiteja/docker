Scenario: Jenkins Pipeline for Docker Hub

Scenario Description

Your DevOps team needs to automate the process of building and deploying a Dockerized web application. Jenkins should be used to build the Docker image from the source code repository and then push the image to Docker Hub. The pipeline must ensure secure authentication using Jenkins credentials and should be reusable for future projects.

Jenkins Pipeline Code


pipeline {
    agent any

    environment {
        DOCKER_HUB_CREDENTIALS = credentials('docker-hub-credentials') 
        IMAGE_NAME = "your-dockerhub-username/your-image-name"
        IMAGE_TAG = "latest"
    }

    stages {
        stage('Checkout') {
            steps {
                checkout scm
            }
        }

        stage('Build Docker Image') {
            steps {
                script {
                    sh "docker build -t ${IMAGE_NAME}:${IMAGE_TAG} ."
                }
            }
        }

        stage('Push to Docker Hub') {
            steps {
                script {
                    sh "echo ${DOCKER_HUB_CREDENTIALS_PSW} | docker login -u ${DOCKER_HUB_CREDENTIALS_USR} --password-stdin"
                    sh "docker push ${IMAGE_NAME}:${IMAGE_TAG}"
                }
            }
        }
    }
}
